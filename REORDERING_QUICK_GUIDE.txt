========================================
MOBILE APP INTEGRATION
========================================

FETCH SECTIONS IN ORDER:
```
SELECT * FROM feature_sections
WHERE is_active = true
ORDER BY display_order ASC;
```

FETCH CATEGORIES IN ORDER:
```
SELECT * FROM categories
WHERE is_active = true
ORDER BY display_order ASC;
```

RENDER HOME SCREEN:
Loop through sections in order:
- If section_type = 'best_seller' ‚Üí Show Best Sellers
- If section_type = 'trending' ‚Üí Show Trending
- If section_type = 'categories' ‚Üí Show Categories
